# wheelie-blog

A personal blog by Tom Easterbrook, sharing personal experiences in adult social
care, local government, and other topics.

This repository hosts a static site.

## .gitignore

A simple `.gitignore` suited to a static site is included:

```gitignore
# Build output
/dist/
/_site/
/public/build/

# Dependencies
/node_modules/

# Environment / secrets
.env
.env.local

# OS / editor cruft
.DS_Store
Thumbs.db
*.swp
.idea/
.vscode/

# Logs
*.log
npm-debug.log*
```
